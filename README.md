# FatsApi login Api

Simple authentication Api using FastApi and SQlite.

## Features

- Register User
- login user
- JWt authentication
- Protected route
- SQlite database
- password hashing

## technologis
- FastApi
- SQlite 
- pydantic
- jwt 
- paslib
